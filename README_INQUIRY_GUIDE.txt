Zhonggu Auto Export 客户询盘查看指南

网站的 Inquiry Form 已接入 Netlify Forms。客户成功提交后，询盘会保存到 Netlify 后台。

查看询盘步骤：
1. 登录 Netlify 后台。
2. 打开对应网站 Site。
3. 进入 Forms。
4. 点击 inquiry。
5. 在 Submissions 中查看客户提交的 Name、Country、WhatsApp、Email、Interested Vehicle、Quantity 和 Message。

询盘统计：
可以在 inquiry 表单的 Submissions 页面导出 CSV 文件，用于客户跟进、车型需求和国家市场统计。

注意事项：
- 网站前端只生成 Sales Reply Draft，不会自动发送邮件或 WhatsApp。
- 回复客户前，请确认库存、价格、运输费用和交付时间。
- 如果 Netlify 未识别表单，请重新部署网站后再提交测试询盘。
